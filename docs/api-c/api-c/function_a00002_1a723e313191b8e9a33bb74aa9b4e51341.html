  • Defined in File GenericContainerCinterface.h

Function Documentation¶

int GC_set_vector_of_complex(double const *re, double const *im, int nelem)¶

Set actual pointed element of GenericContainer to a vector of complex of size nelem and copy vector a of real_type to the element.

Return

error code, 0 = OK

Parameters

  • re: vector of real part to be stored

  • im: vector of imaginary to be stored

  • nelem: number of element of the vector
