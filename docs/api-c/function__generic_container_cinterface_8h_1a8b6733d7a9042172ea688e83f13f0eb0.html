  • Defined in File GenericContainerCinterface.h

Function Documentation¶

int GC_set_vector_of_real(double const *a, int nelem)¶

Set actual pointed element of GenericContainer to a vector of real_type of size nelem and copy vector of real_type a to the element.

Return

error code, 0 = OK

Parameters

  • a: vector of double to be stored

  • nelem: number of element of the vector
