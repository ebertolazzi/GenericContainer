class GenericContainerNamespace::GenericContainer¶

GenericContainer is a class which permit to store eterogeneous data:

  • pointer

  • boolean

  • integer

  • long integer

  • floating point

  • complex floating point

  • string

  • vector of pointer

  • vector of boolean

  • vector of integer

  • vector of floating point

  • vector of complex floating point

  • matrix of floating point

  • matrix of complex floating point

  • vector of string

in addition to this data type the following two container are added

  • vector of GenericContainer

  • map of GenericContainer

GenericContainer const &writeFormattedData(ostream_type &stream, char const delimiter = '\t') const¶

write GenericContainer as regular formatted data

Write the contents of the GenericContainer object to stream.

GenericContainer must be a map which contains the fields:

  • “headers” this element must be a vec_string_type which contains the strings of the headers of the columns of the data

  • ”data” this element must be a vector_type which contais the vectors which are the columns of the data to be saved. Each column can be of type

    1. vec_bool_type

    2. vec_int_type

    3. vec_real_type

    all the vector must have the same size.

Parameters

  • stream: stream to write the output

  • delimiter: desired delimiter (optional). Default is tab.
