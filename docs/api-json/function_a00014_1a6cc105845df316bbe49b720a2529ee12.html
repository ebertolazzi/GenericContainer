Function GC_namespace::jsonStringToGenericContainer¶

Function Documentation¶

void GC_namespace::jsonStringToGenericContainer(std::string const &json, GenericContainer &gc_output, GenericContainer const &gc_options_in)¶

Converts a json string to a GenericContainer. It throws an exception if the parsing of the json string fails. Strings representing pointers are not converted to pointers but remain strings. gc_options is used as above, but the ‘prettified’ and ‘indentation’ options does not bother here.
