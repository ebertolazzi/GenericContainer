Detailed Description

template<typename TYPE>
class GC_namespace::mat_type< TYPE >

Generic matrix storage type.

This template class defines a matrix type that extends vector<TYPE> to store and manipulate a 2D matrix of elements of type TYPE. The matrix is stored internally as a 1D vector in row-major order.

Template Parameters
TYPEThe type of elements stored in the matrix.
