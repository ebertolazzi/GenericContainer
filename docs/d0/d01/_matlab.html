Matlab Support

GenericContainer has an interfacing to exchange data with matlab. To use the interface include

#include "GenericContainerInterface_matlab.hh"
GenericContainerInterface_matlab.hh

compile and link with GenericContainerInterface_matlab.cc. The interface contains a set of functions to convert from GenericContainer to mxArray and the other way around.
